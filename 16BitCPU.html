16-bit 5-Stage RISC CPU Sep 2023 – Mar 2024 • Mostly Completed

This is a custom 16-bit CPU I designed and built as a succesor to my previous 8-bit CISC CPU build.

This was a test of an added memory mapped LCD running at around 140Hz.
Overview

I had reliability issues with by prevous 8-bit CPU, and wanted to make PCB's. Wanting to improve the design I ended up completly redesigning the CPU as a 16-bit CPU with a 5-stage pipeline. The goal of this project was to try and maximize the capability of the CPU, while minimising the chip count. This is most noticible in instruction decoding where I was able to keep the different instruction formats to 7 with room for expansion if needed

  • 5-stage pipeline with data forwarding and hazard resolution
  • Improved ALU supporting bit-wise operations
  • Up to 16 general purpose registers
  • 20 MB of banked addressable memory
  • Seperated instruction and data memory
Lessons & Next Steps

What went well

  • I'm happy with the design of the CPU. I set out to make a pipelined CPU but I also wanted to make it as simple as possible (keeping chip count low, making decode logic as simple as possible)
  • I was able to make a working CPU, although it's not fully complete

What I’d improve

  • Adding a barrel shifter is the biggest improvement I could see.
  • Adding interupts would also improve the design a lot at the cost of a lot of circuitry.

Next steps

  • Create a PCB for the memory stage
  • Continue development
